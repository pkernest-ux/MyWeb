Workflow

維護流程

  1. 點選上方任一 Edit 按鈕。
  2. 在 GitHub 頁面修改文字或內容。
  3. 按 Commit changes 儲存。
  4. 等待 GitHub Actions / Azure 部署完成，通常約 1 到 3 分鐘。
